Calendar Paste

Paste text → preview → create in Google Calendar and Outlook (Microsoft 365). Also supports .ics download for iPhone import.

1) App configuration


Needs Google Calendar scope: https://www.googleapis.com/auth/calendar.events

Needs Microsoft Graph scope: Calendars.ReadWrite
Used for display + event creation.
Google: not connected Outlook: not connected
Tip: Host this on http://localhost:xxxx or HTTPS. OAuth generally won’t work from file://.
